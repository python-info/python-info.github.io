Operatory


Operatory arytmetyczne:

  • dodawanie + oraz odejmowanie -
  • mnożenie *, dzielenie /, dzielenie całkowite //
  • reszta z dzielenia całkowitego %
  • potęgowanie **

Operatory porównania:

  • < (mniejszy niż),
  • > (większy niż),
  • == (równy),
  • != (różny),
  • <= (mniejszy lub równy),
  • >= (większy lub równy),
  • is (sprawdza czy dwie wartości są identyczne),
  • in (sprawdza czy zmienna jest zawarta w innym obiekcie)
  • not (dodaje zaprzeczenie)

Operatory logiczne:

  • and (oznacza, że oba warunki muszą być prawdziwe)
  • or (oznacza, że jeden z dwóch warunków musi być spełniony)

Przykłady:


  >>> 0 < 1
  True
  >>> 0 > 1 
  False
  >>> 0 <= 1
  True
  >>> 0 >= 1 
  False
  >>> 1 == 0
  False
  >>> 1 == 1
  True
  >>> 1 != 0
  True
  >>> 1 != 1
  False  
 


  >>> x = 10
  >>> x < 10 
  False
  >>> x > 1 and x < 13 
  True
  >>> x < 1 and x < 13 
  False
  >>> x > 1 or x < 13 
  True
  >>> x != 10 or x > 13 
  False
  >>> x > 1 and x < 13 or x != 10
  True
